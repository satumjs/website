          SatumJS

          🔱 面向多实例集成、功能可插拔的微前端框架

          快速上手 →

          开箱即用

          只需要一个 @satumjs/core 依赖，配置激活规则后即可看到效果。

          易定制

          微内核架构内置了中间件/插件机制，可通过扩展功能，定制适合的微前端。

          零侵入

          微应用(包括 Vite 应用)不需要任何适配代码，零侵入就能无缝地集成。

          # 安装

          $ yarn add @satumjs/core  # or npm i @satumjs/core -S

          # 使用

          import { register, start } from "@satumjs/core";

// 注册微应用
register({
  name: "reactApp",
  entry: "http://localhost:8000",
  rules: { rule: "/", container: "#mountNode" },
});

// 启动
start();

          或者使用远程的微应用配置，可以直接启动：



           

          import { start } from "@satumjs/core";

start({ remoteUrl: "https://example.com/apps.json" });
